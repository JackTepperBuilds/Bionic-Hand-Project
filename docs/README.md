# 📚 Documentation

This folder contains all documentation for the bionic hand.

Lists lessons learned, challenges faced, demos, etc.

---

## 📁 Contents

### 📦 Bill of Materials (BOM)
Hardware components used in the system, including:
- Raspberry Pi
- Servo motors (MG995)
- PCA9685 servo driver
- Camera module
- Wiring and power components

---

### 🧠 System Reports
Documentation of system versions:

- **MK0 Report**: Initial prototype using MediaPipe-based gesture detection with the raspberry pi camera for vision.
- **MK1 Report**: Refactored code using C++, improved mechanical design, and 17 Degrees of Freedom
